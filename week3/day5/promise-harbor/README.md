# Promise Harbor


### How to Run

1. Save the file as promise-harbor.js.
2. Open a terminal in the project directory.
3. Run the program using: node promise-harbor.js.

View docking results and error handling output in the console.

Features Implemented

Custom dockShip(shipName) function that returns a Promise

Simulated asynchronous arrival using setTimeout

Conditional rejection (invalid name or random failure rate)

Chained .then() calls that transform and pass data forward

Centralized error handling with .catch()

Cleanup/final status messaging using .finally()

Simulation runs for multiple ships concurrently
