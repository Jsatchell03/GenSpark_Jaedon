#Spellbook Manager

This is a simple JavaScript spellbook system that stores and manages spells using objects and arrays.

### How to Run

1. Save the file as spellbook.js.
2. Open a terminal in the project folder.
3. Run the program using: `node spellbook.js`.

### Features Implemented
- Add new spells dynamically with addSpell()
- Print formatted spell details using object destructuring
- Generate spell statistics (total spells, average power, highest power spell)
- Compare spells by power and mana efficiency
