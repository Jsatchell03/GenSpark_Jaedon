# Array Arena

### How It Works
This program prefomrs random opperations on an array of Spencer Dinwiddie's points in his last 10 games.

### Actions/Operations
- addEntry: Adds a random number to the array
- removeEntry: Removes a random entry from the array
- updateEntry: Changes a random entry in the array to a random number
- findEntry: Prints a random entry to the console
- sortRecords: Sorts the array in ascending order
