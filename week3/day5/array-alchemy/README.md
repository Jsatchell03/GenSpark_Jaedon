# Array Alchemy

### How to Run

1. Save the file as array-alchemy.js.
2. Open a terminal in the project directory.
3. Run the program using: `node array-alchemy.js`.


### Features Implemented

- Prints all spells with formatted output using forEach
- Filters spells by power level and element (filter)
- Extracts spell names and calculates mana efficiency (map)
- Generates analytics including total count, averages, and most powerful spell (reduce)
