# Frontend Rescue Ops
## How to Run

```bash
npm install
npm run dev
```

## Features

- Create tasks with a title and priority level (Low / Med / High)
- Three-column board: To Do, Doing, Done
- Move tasks between columns
- Delete tasks with a confirmation prompt
- Duplicate task title warning
- Max 20 task limit
- Priority-colored card indicators
- Responsive dark-themed UI
